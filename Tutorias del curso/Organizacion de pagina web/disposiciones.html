Posicionamiento en CSS

  • static (es por defecto)
  • relative
  • fixed
  • absolute
  • sticky

los div static no permiten editar el Posicionamiento como left, etc
Los div relativos puedo modificarlos y se posicionan en relacion al posicionamiento que tendrian si fuera static (posicion original)
este posicionamiento se usa en barras de navegacion para que
esten abajo del todo o arriba del todo por ejemplo. sin importar los otros elementos
este siempre va a estar ahi

el sticky es una mezcla entre el relativo y el fijo
se usa en menus para darle mas fluides y estetica a los mismos
se mantiene en una posicion pero al hacer scroll modifica su posicion
y pasa de relative a fixed
este es un contenedor que tiene otros div adentro
este siempre va a estar fijo pero respecto a la posicion
del padre (que vendria a ser el elemento mas cercano hacia arriba de el)
puedo poner a su vez este tipo de posicion en un div con posicion relativa
para poder mantenerlo fijo dentro de esa seccion.
